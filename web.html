Yee Mon's Travel Journals

Overview

This Node.js web application enables users to create and manage travel journals effortlessly. Utilizing Express.js, EJS, and Bootstrap, it supports full CRUD operations for documenting and sharing travel experiences with a user-friendly interface

Frontend

  • EJS
  • Bootstrap

Backend

  • Express.js

Project

  • Individual Project
  • Module: Software Application Development
  • Year 2, Semester 1, 2024

Key Features

CRUE Operations

Create (C) : Adding new journal entries.

Read (R) : View and retrieve existing entries from the database.

Update (U) : Edit and Update existing entries.

Delete (D) : Remove or Delete journal entries as needed.

Dynamic Content Rendering

Utilized EJS templating engine to inject dynamic data into HTML templates, allowing for real-time updates to the content displayed on the page based on user interactions and database changes.

Form Validation

Implemented client-side and server-side validation to ensure that users enter valid data, improving user experience and preventing potential errors during data submission.

Responsive Design

Designed the user interface with Bootstrap to ensure that the application is fully responsive, providing an optimal viewing experience on devices of all sizes, from desktops to smartphones.

Final Outcome

Create (C)

Create operation is a feature whereby new journal entries can be added with their respective Location, Travel Period, Title of the journal, Text, Date of the entry and last but not least, an image from the device.

Read (R)

Upon reaching at the main landing page or home page, all the existing journal entries with their specific detailed information and images can be seen within each card.

Update (U)

Update operation provides a feature to freely update each journal information such as changing Texts, Title, Travel period, Date and even Images.

Delete (D)

Delete operation provide a functionality to remove or delete any journal entries. Upon clicking the delete button of each journal, the alert message will appears at the top to validate the choice again.

About Me Page

This is a simple about me page to provide some personal information to the users along with a few engaging pictures.

Contact Page

A contact me page provides an Email, Phone Number and the link to my LinkedIn profile.
